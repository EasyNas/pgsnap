Welcome to the pgsnap Project Home Page

pgsnap is a PostgreSQL tool that mimics orasnap performance report tool for Oracle.

Here are two examples of the HTML in output:

The pgsnapproject is a PostgreSQL Community project that is a part of the pgFoundry.

The pgFoundry page for the project is at http://pgfoundry.org/projects/pgsnap, where you can find downloads, documentation, bug reports, mailing lists, and a whole lot more.
